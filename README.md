## About Books
This provides a list of API endpoints for CRUD operations on a book inventory. Developed using the Laravel framework and MySQL database. It also retrieves books from an external API using the name of the required book

## Getting Started

- Clone the repository.
- Run "composer install".
- Add database configuration to env file.
- Run "php artisan serve" to start development server and you can start using the API endpoints.


## Running tests
- "php artisan test" to run tests
